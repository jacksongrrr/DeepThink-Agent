# 产品愿景

## 项目简介

**DeepThink Agent** 是一个在本地运行的演示型 Web 应用：针对用户提出的开放问题，用 **DeepSeek Chat** 生成多条「思考路径」（每条含短标题与详细阐明），再对**每条路径并行调用 DeepSeek R1** 做分角度推理；随后将各路径的推理与小结汇总，由 **Chat 输出一份综合回答**。支持 **对比模式**：单次纯 R1 与上述「多路径 × 并行 R1 + 综合」管线**同时流式**输出，便于对照效果。界面支持 **Markdown 预览**（内置 marked / DOMPurify），密钥仅通过本机 `.env` 配置，默认面向研究与提示词实验，而非生产多租户服务。

## 目标用户

需要在同一问题下对比「多路径并行 R1 + Chat 综合」与「单次纯 R1」的开发者、提示词工程师与产品研究者。

## 核心场景

1. 用户在 Web 中输入开放问题（如策略、决策、方法论类问题）。
2. 选择运行模式：仅基线（单次 R1）、仅技术（多路径 → 并行 R1 → 综合）、或并排对比（两侧并行流式）。
3. 页面展示：路径列表与阐明、各路径 R1 思考与小结、综合回答；对比模式下同时展示纯 R1 结果（均支持 Markdown 渲染）。

## 成功标准

- 配置 `.env` 后一条命令启动本地站点并可完成端到端调用。
- 三种模式行为清晰、结果可并排阅读。
- 提示词结构符合角色分离、分隔符与可解析输出约定，便于迭代与 A/B。

## 非目标

- 不提供多用户鉴权、持久化对话历史或生产级部署方案（默认本机演示）。
- 不保证模型输出的事实正确性；本项目聚焦流程与对比，而非领域专家系统。
